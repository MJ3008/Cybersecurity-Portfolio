# Random Password Generator (Python)

This is a simple Python project that generates a strong random password.

## Features
- Generates secure random passwords
- User can choose the password length
- Uses Python built-in libraries

## Technologies Used
- Python
- random module
- string module

## How to Run

1. Open terminal
2. Run the following command:

python password_generator.py

3. Enter the desired password length.
